## Image Guidelines

- **Format**: PNG or JPG (PNG recommended for transparency)
- **Size**: 512x512px (square) - will be automatically resized by Next.js
- **Quality**: High resolution for web display
- **Naming**: Use lowercase with underscores (as shown above)
- **Optimization**: Images will be automatically optimized by Next.js Image component

## How to Crop from Poster

1. Open the team poster image in an image editor (Photoshop, GIMP, Canva, etc.)
2. Use the circular selection tool or crop tool
3. Crop each person's photo to include face and upper body
4. Ensure square aspect ratio (equal width and height)
5. Save with the exact filenames listed above
6. Place all files in this `/public/team/` directory

## After Adding Images

Once you've added the images:
1. The website will automatically display them on the Team page
2. Verify at: http://localhost:3000/team
3. Images appear in circular frames with hover effects
